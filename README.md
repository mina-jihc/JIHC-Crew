# JIHC Crew

## App Description
JIHC Crew is a Flutter and Firebase volunteer app built for JIHC students. It helps users discover campus tasks, join crews, chat with teammates, track progress, and manage their profile from one place.

## Features
- Official JIHC email sign-up and login
- Role selection for volunteers and administrators
- Onboarding flow with saved completion state
- Task feed, task details, and task creation
- Crew creation, editing, and management
- Topic-based chat rooms for ecology, sports, and charity
- Progress tracking and leaderboard views
- Profile, notifications, and settings screens
- Firebase Auth, Cloud Firestore, and Storage integration
